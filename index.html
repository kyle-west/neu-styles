Neumorphism

Levels

Neumorphic Design relies on elements living in 3 main levels. Like the levels of a 2-story house, elements reside on the main floor, the top floor, or in the basement.

Interaction

When an element is interacted with, it transitions between all 3 levels. This simulates the pressing of a physical button. Elements that are interactive should have bright/strong colors to attract the user's attention.
